Hyphasia

Getting Started - Pathfinder's Edition

This document describes software in active development.

Current Support Status

OS: Windows

Packaging: Standalone; VST3.

MIDI: 16-channel note on/off with velocity sensitivity, sustain, sostenuto. Partial MPE support.

Prerequisites

The most important thing you should do before using Hyphasia is update your GPU (video card) drivers.

Hyphasia uses only basic hardware features, so you can expect support for some older hardware, but the software requires recent drivers supporting Vulkan 1.3. If your hardware does not have drivers with support for Vulkan 1.3, it will not be possible to run Hyphasia until you upgrade to a GPU with modern driver support.

Installing Hyphasia

All formats: first download the `VC_redist.x64.exe` and install it. You only need to do this once on each machine--even if you download a new version of Hyphasia later.

Standalone: download the `.exe` anywhere convenient and double-clicked to launch. No specific steps are required to install it. Windows may warn you about unsigned code. Click ‘More info → Run anyway’.

VST: download the `.vst.zip` file and unzip it into your system VST directory (often `C:\Program Files\VstPlugins\`, but it may depend on your DAW). You should unzip the entire directory (with name ending in `.vst`) as-is.
