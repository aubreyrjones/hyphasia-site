Installing Hyphasia

  1. Download the `VC_redist.x64.exe` and install it. You only need to do this once, ever--even if you download a new version of Hyphasia later.
  2. The standalone `hyphasia.exe` can simply be downloaded anywhere and double-clicked to launch. No specific steps are required to install it.
  3. To install the VST version, unzip it into your system VST directory (often `C:\Program Files\VstPlugins\`, but it may depend on your DAW).
