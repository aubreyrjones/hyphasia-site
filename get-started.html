Hyphasia

Getting Started - Pathfinder's Edition

This document describes software in active development.

Current Support Status

OS: Windows

Packaging: Standalone; VST3i.

MIDI: Note on/off with velocity sensitivity.

Prerequisites

The most important thing you should do before using Hyphasia is update your GPU (video card) drivers.

Hyphasia uses only basic hardware features, so you can expect support for older high-end hardware, but the software requires recent drivers supporting Vulkan 1.3. If your hardware does not have drivers with support for Vulkan 1.3, it will not be possible to run Hyphasia until you upgrade to a GPU with newer driver support.

Installing Hyphasia

  1. Download the `VC_redist.x64.exe` and install it. You only need to do this once on each machine--even if you download a new version of Hyphasia later.
  2. The standalone `hyphasia.exe` can simply be downloaded anywhere and double-clicked to launch. No specific steps are required to install it.
  3. To install the VST version, unzip it into your system VST directory (often `C:\Program Files\VstPlugins\`, but it may depend on your DAW). You should unzip the entire directory (with name ending in `.vst`) as-is.
